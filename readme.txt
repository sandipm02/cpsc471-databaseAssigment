# Installation

Download the entire github contents to a folder where you desire. 
Import the studyhubdb.sql into a mysqlworkbench (local instance) server
Navigate to folder in a terminal
Type “node server.js” and run
Open the index.html (or any other file) in the browser of your choice and the website will be functional 


# Packages

The project uses node js, mysql, axios, mysqlstring, express, and body-parser.
